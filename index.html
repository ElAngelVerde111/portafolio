Sobre mí

Soy Angel Verde, desarrollador web con enfoque en proyectos experimentales, ocultistas y sistemas de alto impacto visual y filosófico.

Trabajo combinando:

  • Backend con Python / Flask, PHP, Node.js, Ruby, Wordpress.
  • Frontend con Vue y React. Interfaces crudas, estéticas místicas, y performance sólida.
  • Desarrollo de CLI Tools personalizadas para automatización y control total
  • Algoritmos de IA y lógica cuántica
  • Seguridad informática y anonimato digital
  • Especializado en aplicaciones decentralizadas

Mi enfoque se basa en la estética underground, la descentralización y la creación de software fuera de las reglas tradicionales.
